# CinemaScore: Movie Review & Rating Platform
A dynamic web application that allows users to browse movies, read community reviews, and submit their own ratings.

## Technical Highlights
* **Relational Database:** Designed a SQL schema to manage many-to-many relationships between Users, Movies, and Reviews.
* **CRUD Operations:** Implemented full Create, Read, Update, and Delete functionality for review management.
* **Responsive Design:** Used HTML5 and CSS3 to ensure the site is accessible across mobile and desktop devices.
